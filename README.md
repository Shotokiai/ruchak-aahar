# Aahar Landing Page

A lightweight Next.js landing page with a swipeable onboarding card flow.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the dev server:
   ```bash
   npm run dev
   ```

## Notes

- Replace the placeholder SVG images in `public/illustrations/` with your provided artwork.
- This page is built as a single-screen swipeable card flow matching the requested design.
